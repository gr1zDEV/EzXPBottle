# XPBottle Documentation

Welcome to **XPBottle** by **EzInnovations**.

XPBottle allows players to convert exact raw XP into custom XP bottle items, then redeem the same exact value later. It supports command-based usage and a configurable GUI flow.

## Feature Summary

- Exact XP withdraw and redeem (raw XP points, not random vanilla XP).
- Configurable `/xpbottle` GUI with preset buttons, info panel, withdraw-all button, and close button.
- Custom item identification through `PersistentDataContainer` metadata.
- Redeem support from the clicked hand (main hand or offhand), including optional shift-right-click redeem-all for the in-hand stack.
- Admin bottle-give and config reload commands.
- Optional PlaceholderAPI placeholders.

## Compatibility

XPBottle is built for modern **Paper** servers and is designed with **Folia-safe** scheduling assumptions in mind.
